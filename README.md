# Jar Search

A simple Go command line application to allow maven central based searches for Jar files.

## Setup

```
gom install
gom test
```

## Build and run

* Gradle: `gom run jarsearch.go gradle <JAR_NAME>`
* Maven: `gom run jarsearch.go maven <JAR_NAME>`

## Install

`gom install jarsearch.go`
